Juice Box Home Page

Routes:

    Users

  • Get all users: [method: GET, route: /api/users]
  • Get user by id: [method: GET, route: /api/users/:userId]
  • Login [method: POST, route: /api/users/login, body: [username, password]]
  • Register: [method: POST, route: /api/users/register, body: [username, password, name, location]]

    Posts

  • Get all posts: [method: GET, route: /api/posts]
  • Create Post [method: POST, route: /api/posts, body: [title, tags, content]]
  • Edit post by id: [method: PATCH, route: /api/posts/:postId, [title, tags, content]]]
  • Delete Post: [method: DELETE, route: /api/posts/:postId]

    Tags

  • Get all tags: [method: GET, route: /api/tags]
  • Get posts by tag name: [method: GET, route: /api/:tagName/posts]
